# Fizzy

Desktop virtual pet that receives notifications from AI coding agents (Claude Code, Codex, etc.) and displays them as an interactive bubble companion on your screen.

## Quick Start

```bash
swift build
swift run Fizzy
```

Fizzy listens on `localhost:7319` for notification payloads at `POST /claudecode/notification`.

## Test

```bash
swift test
```
